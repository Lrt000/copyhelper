抄写助手 - 分词句朗读工具

📖 简介

抄写助手是一不用看着屏幕的听写朗读工具。它可以将文章按词句分割，逐词停顿朗读适合抄作文。

✨ 核心功能

🎯 两种朗读模式

| 模式 | 说明 |
|------|------|
| **普通模式** | 按标点符号（。，；？！）分句朗读 |
| **AI分词模式** | 按语义智能分词，支持自定义停顿标记 |

🎮 核心特性

- **逐词停顿朗读** - 每个词语之间有可调节的停顿，便于跟读
- **快捷键控制** - 支持自定义快捷键（上一句/重读/下一句/暂停）
- **参数可调** - 语速（0.65x-1.25x）、音量（0.5-3倍，但是我测试后没有明显感觉到。无所谓了，声音调大点就行）、停顿时长
- **深色模式** - 护眼的深色界面，长时间使用不疲劳

🤖 AI分词模式

利用大语言模型LLM（随便什么都行）实现智能分词：
1. 粘贴文章 → 2. 复制提示词 → 3. AI返回标记结果 → 4. 粘贴朗读

标记规则：

- `**` 词间停顿（短停顿）
- `##` 分句/段落停顿（长停顿）

🎮 快捷键说明

| 功能 | 默认键 | 说明 |
|------|--------|------|
| 上一句 | `A` | 可自定义 |
| 重读当前句 | `S` | 可自定义 |
| 下一句 | `D` | 可自定义 |
| 暂停/继续 | `空格` | 固定快捷键 |
> 点击快捷键输入框后按下任意键即可自定义。

📝 使用教程

**普通模式**
1. 在输入框粘贴文章
2. 点击「开始朗读」
3. 使用快捷键或按钮控制播放

**AI分词模式**
1. 切换到「AI分词模式」
2. 粘贴文章，点击「复制提示词」
3. 将提示词发给AI（示例提示词已内置）
4. 将AI返回的结果粘贴回来
5. 点击「开始朗读」

⚙️ 参数设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 词语间停顿 | 1000ms | 每个词之间的间隔 |
| 句子间停顿 | 1200ms | 句子之间的间隔 |
| 语速 | 1x | 0.65x-1.25x可选 |
| 音量 | 1倍 | 0.5-3倍可选 |

❓ 问题

**Q: 支持手机使用吗？**
A: 不支持，貌似是手机端无法用TTS合成语音，要做只能使用在线的，但是我懒得去搞了，所以手机目前是不支持的。

**Q: 为什么有时候复制提示词失败？**
A: 部分浏览器需要HTTPS环境才支持剪贴板API。如果失败，会弹出提示框供手动复制。
